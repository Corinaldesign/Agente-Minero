1. INTRODUCCIÓN
===========================================
El Agente Minero es un sistema inteligente diseñado para el análisis y procesamiento de datos en el sector minero. Este documento registra el desarrollo, cambios y mejoras implementadas en el sistema.

===========================================
2. ESTRUCTURA DEL PROYECTO
===========================================
/src
  /components      - Componentes React
  /services        - Servicios de API y procesamiento
  /models          - Modelos de IA y análisis
  /utils           - Utilidades y helpers
  /styles          - Estilos y temas
  /tests           - Tests unitarios y de integración

Archivos principales:
- main.py         - Punto de entrada backend
- App.tsx         - Componente principal React
- requirements.txt - Dependencias del proyecto
